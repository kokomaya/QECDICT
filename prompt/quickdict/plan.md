# QuickDict — 基于 ECDICT 的屏幕取词翻译工具

## 1. 项目概述

基于 ECDICT 词典数据库，开发一款 Windows 平台的屏幕取词翻译工具。用户通过快捷键唤起取词模式，工具自动识别鼠标位置下的英文单词并弹出美观的翻译弹窗。

---

## 2. 核心功能

### 2.1 快捷键唤起
- **连按 Ctrl 键唤起**：监听键盘事件，检测 500ms 内连按两次 Ctrl 键激活取词模式
- **备选方案**：支持自定义快捷键（如 `Ctrl+Alt+D`）
- 唤起后进入"取词模式"，鼠标移动到单词上即显示翻译弹窗
- 再次按 Esc 或点击弹窗外区域退出取词模式

### 2.2 屏幕取词
- **方案 A — UI Automation（推荐首选）**：使用 Windows UI Automation API 获取鼠标下方的文本控件内容，从中提取鼠标位置对应的单词。适用于浏览器、编辑器、Office 等标准控件
- **方案 B — 剪贴板辅助取词**：模拟双击选中单词 + `Ctrl+C` 复制到剪贴板，再读取剪贴板内容。简单但有侵入性
- **方案 C — OCR 屏幕识别**：截取鼠标周围小区域屏幕截图，使用 OCR（如 Windows.Media.Ocr / Tesseract / RapidOCR）识别文字，提取最近单词。通用性最强，适用于图片、PDF、游戏等场景
- **最终策略**：A + C 组合，优先 UI Automation，失败时回退到 OCR

### 2.3 词典查询
- 利用 ECDICT 的 SQLite 数据库 + `stardict.py` 的 `StarDict` 类进行查询
- 首先将 `ecdict.csv` 转换为 SQLite 数据库（`ecdict.db`），利用已有的 `stardict.py` 工具
- 查询流程：原词查询 → 词形还原（利用 exchange 字段的 lemma 信息）→ 模糊匹配

### 2.4 翻译弹窗（Popup）
- 弹窗跟随鼠标位置显示，自动避免超出屏幕边界
- 显示信息：
  - **单词** + **音标**（phonetic）
  - **中文释义**（translation）
  - **英文释义**（definition）
  - **柯林斯星级**（collins，用 ★ 图标展示）
  - **词频信息**（BNC / 当代语料库排名）
  - **考试标签**（tag：四六级 / 考研 / 雅思 / GRE 等）
  - **词形变化**（exchange：过去式、分词等）
- 点击弹窗外区域或按 Esc 关闭

## 3. 技术架构

### 3.1 技术栈
| 组件         | 技术选型                    | 说明                              |
| ------------ | --------------------------- | --------------------------------- |
| 语言         | Python 3.10+                | 与 ECDICT 原生兼容               |
| UI 框架      | PyQt6 / PySide6             | 现代化 UI，支持自定义样式和动画   |
| 系统托盘     | QSystemTrayIcon             | 常驻后台，托盘图标管理            |
| 键盘监听     | pynput                      | 全局键盘钩子，监听快捷键          |
| 鼠标位置     | pyautogui / ctypes          | 获取鼠标坐标                      |
| 屏幕取词     | uiautomation + rapidocr     | UI Automation + OCR 双模式        |
| 词典引擎     | sqlite3 + stardict.py       | 复用 ECDICT 已有的查询接口        |
| 打包分发     | PyInstaller / Nuitka        | 打包为独立 exe                    |

### 3.2 项目结构
```
quickdict/
├── main.py                 # 程序入口，初始化各模块
├── app.py                  # QApplication 与系统托盘管理
├── hotkey.py               # 快捷键监听（连按 Ctrl 检测）
├── word_capture.py         # 屏幕取词模块（UI Automation + OCR）
├── dict_engine.py          # 词典查询引擎（封装 StarDict）
├── popup_widget.py         # 翻译弹窗 UI 组件
├── overlay_widget.py       # 取词模式的全屏半透明覆盖层
├── styles/

├── assets/
│   ├── icon.png            # 托盘图标
│   └── sound/              # 发音音频（可选）
├── config.py               # 配置管理（快捷键、主题等）
├── utils.py                # 工具函数
└── requirements.txt        # 依赖清单
```

### 3.3 模块交互流程
```
用户连按 Ctrl
    ↓
hotkey.py 检测到激活信号
    ↓
overlay_widget.py 显示半透明覆盖层（表示取词模式激活）
    ↓
鼠标移动 → word_capture.py 获取鼠标下的单词
    ↓
dict_engine.py 查询 ECDICT 数据库
    ↓
popup_widget.py 在鼠标位置附近弹出翻译气泡
    ↓
用户按 Esc / 点击外部 → 关闭弹窗，退出取词模式
```

## 4. UI 设计

### 4.1 弹窗样式
- **圆角卡片**：8px 圆角，带轻微阴影（drop shadow）
- **配色方案**：
  - 浅色模式：白底 `#FFFFFF`，主文字 `#1A1A2E`，次要文字 `#6C757D`，强调色 `#4361EE`
  - 深色模式：深底 `#1E1E2E`，主文字 `#E0E0E0`，强调色 `#7C83FD`
  - 跟随系统深浅色模式自动切换
- **动画效果**：弹窗弹出时带淡入 + 轻微上滑动画（150ms）
- **字体**：英文使用 Inter / Segoe UI，中文使用 Microsoft YaHei UI
- **布局示例**：

## 5. 词典引擎细节

### 5.1 数据库准备
- 首次启动将 `ecdict.csv` 导入 SQLite 数据库 `ecdict.db`
- 利用 `stardict.py` 的 `StarDict` 类和 CSV 导入工具
- 建立额外索引以加速模糊查询和词形还原

### 5.2 查询策略
```
输入单词
  ↓
1. 精确匹配 word 字段（大小写不敏感）
  ↓ 未命中
2. 去除词尾变化，尝试还原词形（利用 exchange 字段的 "0" lemma 映射）
  ↓ 未命中  
3. stripword 处理后尝试模糊匹配
  ↓ 未命中
4. 使用 match() 方法返回最接近的候选词列表
```

### 5.3 词形还原增强
- 构建一张 `lemma` 反查表：从所有词条的 `exchange` 字段中提取变形 → 原形映射
- 存入单独的 SQLite 表或内存字典，加速变形词的查询
- 例：用户取到 "perceived" → 通过 lemma 表找到原形 "perceive" → 返回 perceive 的完整释义

## 6. 优化要点

### 6.1 性能优化
- **数据库连接池**：保持 SQLite 连接常驻，避免每次查询重新打开
- **内存缓存（LRU）**：对最近查过的 500 个单词做 LRU 缓存，重复取词秒出结果
- **延迟查询**：鼠标悬停 200ms 后才触发取词，避免鼠标快速移动时大量无效查询
- **异步查询**：数据库查询在后台线程执行，不阻塞 UI 线程
- **OCR 区域最小化**：仅截取鼠标周围 200×60 像素区域做 OCR，提升识别速度

### 6.2 取词准确性优化
- **单词边界检测**：根据鼠标 X 坐标在识别文本中定位最近的单词，使用空格/标点作为分隔
- **词组识别**：识别常见词组（如 "look forward to"、"make up"），可扩展词组词典
- **驼峰/下划线拆分**：对代码中的 `camelCase` 和 `snake_case` 自动拆分为独立单词查询
- **数字/符号过滤**：过滤纯数字和特殊符号，只对英文字母序列进行查询

### 6.3 用户体验优化
- **弹窗位置智能调整**：弹窗默认出现在鼠标下方偏右，如接近屏幕边缘则自动翻转方向
- **弹窗固定模式**：点击弹窗上的 📌 图标可将弹窗固定，不随鼠标移动消失
- **朗读发音**：集成 TTS（pyttsx3 或 Windows SAPI）朗读单词
- **复制释义**：单击释义文本可快速复制到剪贴板
- **查词历史**：记录最近查过的单词，支持回顾
- **收藏生词本**：在弹窗中增加 ⭐ 收藏按钮，将生词保存到本地生词本
- **多显示器支持**：正确处理多显示器、不同 DPI 缩放的情况

### 6.4 可扩展功能（后续版本）
- **划词翻译**：选中文本后自动弹出翻译（除快捷键取词外的第二种模式）
- **句子翻译**：选中整句时调用在线翻译 API（如百度/有道翻译 API）
- **单词本导出**：导出收藏的生词为 Anki 卡片（.apkg）或 CSV
- **在线词典联动**：弹窗中增加"在线查询"按钮，跳转到有道/Collins 在线词典
- **OCR 语言扩展**：未来支持日语、韩语等其他语言的屏幕取词
- **插件系统**：允许用户自定义取词后的处理逻辑（如写入笔记软件）
- **自动更新**：检测新版本并支持自动更新
- **全局搜索框**：`Ctrl+Alt+S` 打开独立搜索框，直接输入单词查词（不依赖屏幕取词）
